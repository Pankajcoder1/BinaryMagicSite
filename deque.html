  1. What is deque ?
    Deque stands for Double ended queue. It is a sequence container that supports insertion and deletion of elements from both ends with constant time complexity
    Use vector inside deque syntax
    deque<vector<int>> myDeque;
    vector<int> v = {1, 2, 3, 4, 5};
    myDeque.push_back(v);
