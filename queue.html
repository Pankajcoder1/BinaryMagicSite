  1. What is queue ?
    Queue container is implementation of queue data structure, which follows the FIFO (First In First Out) principle.
    Difference between stack and queue ?
    Stack follow LIFO (last in first out) principle where as queue follow FIFO (first in first out) principle.
    Main operation on stack container
    • size(): return size of stack.
    • empty(): check if stack is empty or not, return boolean value accordingly.
    • front(): return front element of stack.
    • back(): return back element of stack.
    • push(element): add a element to top of stack.
    • pop(): erase top element of stack.
    Use vector inside queue, syntax
    queue<vector<int>> q;
    vector<int> v = {1, 2, 3, 4, 5};
    q.push(v);
    vector<int> v2 = q.front();
