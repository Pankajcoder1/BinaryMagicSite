  1. What is priority queue ?
    Priority Queue container is implementation of heap data structure, which place highest/lowest (based of types) priority element at top.
    Main operation on priority queue container
    • size(): return size of priority queue.
    • empty(): check if priority queue is empty or not, return boolean value accordingly.
    • top(): return top element of priority queue.
    • push(element): add a element to top of priority queue.
    • pop(): erase top element of priority queue.
    Syntax for max heap
    priority_queue<int> pq;
    Syntax for min heap
    priority_queue<int, vector<int>, greater<int>> pq;
