  1. About stack container
    STL provide stack container which is just implementation of stack data structure, which allow to perform LIFO operation on it.
    Main operation on stack container
    • size(): return size of stack.
    • empty(): check if stack is empty or not, return boolean value accordingly.
    • top(): return top element of stack.
    • push(): add a element to top of stack.
    • pop(): erase a element from top of stack.
